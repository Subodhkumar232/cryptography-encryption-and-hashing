## Encryption

Encryption is a method of protecting data by converting it into a secure form.

### Symmetric Encryption

* Uses one key for encryption and decryption.
* Faster and commonly used for large amounts of data.

Examples:

* AES
* DES

### Asymmetric Encryption

* Uses two keys: a public key and a private key.
* More secure for sharing information.

Examples:

* RSA
* ECC

## Hashing

Hashing converts data into a fixed-length value called a hash. It is mainly used for password protection and data verification.

Examples:

* SHA-256
* SHA-512
