# Cryptography Report

## Introduction

Cryptography is used to protect information and keep data secure. It plays an important role in cybersecurity.

## Symmetric Encryption

Symmetric encryption uses the same key to encrypt and decrypt data. It is fast and easy to use.

Examples:

* AES
* DES

## Asymmetric Encryption

Asymmetric encryption uses two different keys: a public key and a private key. It provides secure communication.

Examples:

* RSA
* ECC

## Hashing

Hashing converts data into a unique fixed-length value called a hash. It is commonly used for password storage and data integrity checks.

Examples:

* SHA-256
* SHA-512
